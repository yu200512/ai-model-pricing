# AI Token Price — 技能详情描述（扣子商店版）

---

AI Token Price 是一个专注 AI 大模型 API 定价的查询工具。我是这个领域的独立开发者，同时运营开源项目 ai-model-pricing，每周自动抓取各厂商官方定价页。目前已持续追踪 DeepSeek、智谱AI、通义千问、Kimi、星火大模型、百川智能、MiniMax、小米、腾讯云混元、字节豆包、OpenAI、Anthropic、xAI/Grok、Google Gemini 在内的 14 家主流 AI 厂商、数百个模型的定价变动，致力于为需要选择 AI API 的开发者提供透明、实时的价格参考。

核心功能覆盖四大场景。**价格查询**：输入模型名称或选择供应商，快速查看输入-token、输出-token及缓存-token的当前定价，同时支持人民币和美元双币种换算。**场景推荐**：按任务类型（翻译、长文分析、代码生成、推理等）智能推荐性价比最优的 TOP 3 模型组合。**模型对比**：并排对比任意两个模型的定价结构，直接输出价格差异结论。**全量列表**：一览所有收录模型的价格信息，方便横向比较。所有数据覆盖国内外 13 家主流 AI 厂商，国内模型直连查询，国外模型数据每日从 GitHub 同步更新。

使用非常简单，直接用自然语言描述你的需求即可，无需记住特定指令格式。例如：「DeepSeek-V4-Pro 多少钱」「对比 GPT 和 Claude 的价格」「翻译任务推荐便宜且效果好的大模型」「列出所有国产模型价格」。技能会自动识别你的意图并返回对应结果。

几点说明：所有价格数据均以各厂商官方定价为准，本技能仅作参考；美元与人民币换算采用固定参考汇率 1 USD ≈ 7.2 CNY；国外模型价格数据来自 GitHub 每日快照，非实时直连，存在一定更新延迟；部分采用非 token 计费模式的模型已在数据中另行标注。请在做出采购决策前，前往各厂商官网确认最新报价。
