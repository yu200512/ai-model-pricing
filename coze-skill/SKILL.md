# AI Token Price

## 一句话简介
查询主流AI模型价格并推荐高性价比方案。

## 详细介绍

AI Token Price 用于查询 DeepSeek、智谱AI、Qwen、Kimi、星火、百川、MiniMax、小米、腾讯云、字节豆包，以及 OpenAI、Anthropic、Google Gemini、xAI/Grok 等 14 家主流供应商的 API 定价。

技能支持：

- **价格查询**：输入模型名或供应商名，返回输入价、输出价、缓存价、计费单位和上下文信息。
- **模型对比**：输入两个模型，按统一美元口径展示并列价格，便于快速判断成本差异。
- **场景推荐**：按文本、推理、代码、视觉、语音等场景推荐性价比 TOP 3。
- **全量列表**：按供应商分组列出当前价格库中的模型。

数据策略：

- 国内供应商优先探测官方定价页可用性，失败时自动回退 GitHub JSON 数据。
- 国外供应商仅使用 GitHub JSON 数据，并在回复中标注来源和更新时间。
- 人民币价格会按 `1 USD ≈ 7.2 CNY` 同时展示美元换算，便于横向比较。

## 使用案例

### 案例 1：查询单个模型价格

**用户输入：**

```text
查一下 DeepSeek-V4-Flash 多少钱
```

**技能回复：**

```markdown

- 供应商：DeepSeek
- 上下文：1,000,000 tokens
- 类型：reasoning
| 计费 | 输入 | 输出 | 统一价 | 缓存命中 |
|---|---:|---:|---:|---:|
| 未注明 | $0.14/1M tokens | $0.28/1M tokens | — | $0.0028/1M tokens |
```

## 注意事项

1. 价格会随供应商活动、区域、账户等级和批处理折扣变化，最终账单以官方平台为准。
2. 技能中的美元换算使用固定参考汇率 `1 USD ≈ 7.2 CNY`，不代表实时外汇牌价。
3. 部分图像、视频、语音、搜索、存储类服务不是 token 计费，比较时会单独显示计费单位。
4. 国内官方页面可能存在登录、动态渲染或网络超时，技能会自动回退到 GitHub JSON。
5. 国外供应商按上架设计仅从 GitHub JSON 读取，不直接抓取官方页面。
