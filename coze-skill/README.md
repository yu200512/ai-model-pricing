# AI Token Price

查询全球主流 AI 大模型 API 定价，支持比价和场景推荐。

## 功能

- **价格查询**：按模型名或供应商查询当前 API 定价
- **场景推荐**：按任务类型（翻译、代码、推理等）推荐最具性价比模型
- **模型对比**：并排对比两个模型的价格差异
- **全量展示**：按供应商分组列出所有模型定价

## 数据覆盖

| 区域 | 供应商 | 数据来源 |
|------|--------|---------|
| 国内 | DeepSeek、智谱AI、Qwen、Kimi、星火、百川、MiniMax、小米、腾讯云 | 官方页面直连 + GitHub 兜底 |
| 国外 | OpenAI、Anthropic、xAI/Grok、Google Gemini | GitHub JSON（每日更新） |

## 使用示例

- "DeepSeek-V4-Pro 现在多少钱"
- "翻译任务用哪个模型最便宜"
- "GPT-5.4 和 Claude 哪个划算"
- "列出所有国内模型"

## 技术架构

```
用户提问 → 意图识别 → 国内直连/GitHub兜底 → Markdown表格回复
```

扣子 Python 环境，依赖仅标准库（`urllib` + `json` + `re`），无需额外安装。
